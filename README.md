# Интерпретатор расширенной обратной польской записи (ЯОПЗ)

## Описание проекта

Данный проект представляет собой интерпретатор второго внутреннего представления программ на основе **расширенной обратной польской записи (ЯОПЗ)**.

Интерпретатор реализует **виртуальную стековую машину**, предназначенную для выполнения промежуточного кода, сгенерированного компилятором высокоуровневого языка программирования.

Проект ориентирован на изучение принципов:
- работы стековых виртуальных машин
- выполнения промежуточного представления программ
- интерпретации байт-кода
- организации памяти и управления потоком выполнения

---

## Основные возможности

### Арифметические операции
- Сложение
- Вычитание
- Умножение
- Деление
- Остаток от деления

### Операции сравнения
- Больше (`>`)
- Меньше (`<`)
- Равно (`==`)
- Не равно (`!=`)
- Больше или равно (`>=`)
- Меньше или равно (`<=`)

### Управление потоком выполнения
- Условный переход (`Z`)
- Безусловный переход (`J`)
- Вызов подпрограмм (`%`)
- Возврат из подпрограмм (`R`)

### Работа с памятью
- Присваивание значений (`@`)
- Индексация массивов (`E`)
- Работа с константами (`C`)
- Работа с переменными (`I`)

### Ввод-вывод
- Ввод данных (`I`)
- Вывод данных (`O`)

### Управление данными
- Поддержка целочисленных переменных
- Поддержка массивов

## Архитектура системы

Проект состоит из следующих компонентов:

- **Виртуальная машина**  
  Выполняет байт-код ЯОПЗ

- **Стековая память**  
  Используется для хранения промежуточных результатов вычислений

- **Таблицы символов**
  - Таблица констант (`test.cta`)
  - Таблица меток (`test.lab`)
  - Таблица функций (`test.fun`)
  - Таблица переменных и массивов

---

## Форматы файлов

- `test.asa` — основной файл с байт-кодом ЯОПЗ (последовательность команд)
- `test.cta` — таблица констант
- `test.lab` — таблица меток для переходов
- `test.fun` — таблица функций

---

## Пример использования

Пример программы, вычисляющей сумму чисел от 1 до `N`:

1. Пользователь вводит число `N`
2. Программа вычисляет сумму:  
   `1 + 2 + ... + N`
3. Результат выводится на экран
